## 本地阅读

> [!NOTE]
> 请先安装 [Node.js](https://nodejs.org/zh-cn) 和 npm

1. 将此仓库克隆至本地
   `git clone https://github.com/MSCPO/ServerDocumentation.git`
2. 移动至仓库根目录
   `cd ServerDocumentation`
3. 安装依赖（国内环境使用 [cnpm](https://npmmirror.com/) 会更快）
   `npm install`
4. 本地测试
   `npm run dev`
5. 浏览器打开
   `http://localhost:5173/`
